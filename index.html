会社情報スクレイピングAPI

指定した会社のWebページから電話番号/メールアドレス/お問い合わせリンクをスクレイピングします。主に営業リストの作成を目的としております。 必ずしも取得した連絡先が適切であることは保証できないため、あくまで営業リスト作成にあたっての補助ツールとして利用していただければと思います。

フォーム入力

以下に会社のホームページURLを入力することで、会社の電話番号/メールアドレス/お問い合わせリンクを取得できます。

スプレッドシート更新

以下の手順でスプレッドシートを設定することで、スプレッドシートの読取/更新ができます。

  1. 新しいスプレッドシートの作成
    営業リストなどが存在する場合は、新しくスプレッドシートを作成することを推奨します。 スプレッドシートのA列2行目に会社HPのURLを貼り付けます。 ※1行目はヘッダーとして使用するため、データは2行目から貼り付けてください。
  2. スプレッドシートIDの取得
    スプレッドシートIDはGoogleスプレッドシートのURLの一部です。以下の位置にIDがあります。
    
https://docs.google.com/spreadsheets/d/{スプレッドシートID}/edit?gid=0#gid=0
  3. データの出力
    以下にスプレッドシートIDを入力することで、スプレッドシートのB列に電話番号/C列にメールアドレス/D列にお問い合わせリンクが出力されます。

URLからのJSONデータ取得

次のURLに会社のホームページURLをクエリパラメータとして追加してください。


GET https://clientinfo.vercel.app/extract?company_url={会社のHPのURL}

例えば、https://connect-world.co.jp/を指定する場合は以下のようになります。


GET https://clientinfo.vercel.app/extract?company_url=https://connect-world.co.jp/

APIは以下のようなJSONデータを返します。

            

{
    "https://connect-world.co.jp/": {
        "phone_numbers": ["+81 90-1234-5678"],
        "emails": ["info@connect-world.co.jp"],
        "contact_links": ["/contact"]
    }
}
