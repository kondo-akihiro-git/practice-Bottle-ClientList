会社情報スクレイピングAPI

このAPIは、指定したURLのWebページから電話番号、メールアドレス、お問い合わせリンクを取得し、JSON形式で返却します。

使い方

以下のエンドポイントにPOSTリクエストを送信してください。

    POST /extract
    Body: {
    "urls": [
        "https://example.com",
        "https://anotherexample.com"
    ]
    }

戻り値

例:

    {
    "https://example.com": {
        "phone_numbers": ["+81 90-1234-5678"],
        "emails": ["info@example.com"],
        "contact_links": ["/contact"]
    }
    }
